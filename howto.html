üîó Submitting Your Code via GitHub

Once you've created your Pookkalam design using the language of your choice, the next step is to submit it. For this competition, submissions are done through GitHub. Don't worry if you're new to GitHub ‚Äî here's a quick guide to help you get started.

Step 1: Create a GitHub Account
If you don't already have a GitHub account, head over to GitHub.com and sign up for a free account.

See how: https://youtu.be/QUtk-Uuq9nE?si=ECSPHNIEDmu9IFME

Step 2: Create a New Repository
After logging in, click on the + icon in the top right corner and select New repository.
Name your repository Code-a-Pookkalam-2024.
Choose the repository as Public so that we can access your submission.
Add a brief description like "My Code-a-Pookkalam submission."
Optionally, initialize the repository with a README.md file.

Step 3: Upload Your Code
If you are familiar with Git, you can use git commands to push your code to GitHub.
Alternatively, you can manually upload your code by:
Clicking on Add file in your repository.
Selecting Upload files and then dragging your code and any images into the file uploader.
Don't forget to include a screenshot of your final output in your repository.

See how: https://youtu.be/P75e8DgOxn8?si=FK7ptPAWEaWELMvu

Step 4: Submit Your Repository Link
Once your code is uploaded to GitHub, submit the link to your repository using the form provided on the competition page.

Example format for the link:

https://github.com/YourUsername/Code-a-Pookkalam-2024
